# Gateway devices Service

## Running APP

1. install modules
```bash
$ npm install
```
2. take copy from `.env-example` and change name of copy file to `.env` and add your data
3. start the App

```bash
# development
$ npm run start

# update entities in dist
$ npm run build

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Running Test

```bash
npm run test
```

## Running Lint

```bash
npm run lint
```

## Running Seed

npm run seed
